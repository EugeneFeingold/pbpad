Overlays · triggered by hardware & idle, not the menu

Dim

Half brightness after the Screen Dim timeout. Any input wakes it.

Sleep

Screen off after Screen Off. Polling pauses. “Touch any control to wake.”

Locked

Hold both knobs to lock or unlock — pocket-proofing.

Shutting down

Hold the power button 5s → “Hold to shut down” → clean halt.
